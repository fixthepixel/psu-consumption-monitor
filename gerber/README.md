# Gerber Fabrication Files

This directory contains **generated Gerber files** for the
**PSU Consumption Monitor** PCB.

These files are provided for convenience to allow users to
manufacture the board without modifying the KiCad project.

---

## ⚠️ Important Notes

- The **KiCad schematic and PCB files** are the authoritative source.
- Gerber files are **derived artifacts** and may become outdated.
- Always review the files in a Gerber viewer before ordering PCBs.

---

## 📐 Board Information

- Layer count: 4
- Copper layers: F_Cu / In1_Cu / In2_Cu / B_Cu
- Separate drill files for PTH and NPTH
- Includes modern `.gbrjob` file
